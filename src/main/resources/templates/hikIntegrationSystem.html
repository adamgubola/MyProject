Hikvision Rendszer Integráció

Fejlesztés alatt

Ez a modul jelenleg fejlesztés alatt áll React alapokon.

Mivel a fejlesztői környezetben vagyunk, a Java szerver (8080) és a React kliens (3000) külön fut.

Ugrás a React Felületre (Port 3000)

Itt fog megjelenni a riasztórendszer térképe és kezelőfelülete.
